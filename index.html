🔗 국토교통부 실거래가 API 연동

합법적이고 정확한 공공데이터를 활용해 실시간 시세를 가져올 수 있습니다.

📌 안내: 공공데이터포털에서 무료로 API 키를 발급받으실 수 있습니다. 발급에 1~2일 소요됩니다.
1 공공데이터포털 회원가입

🌐 www.data.go.kr 접속 → 회원가입 (무료)

2 ⚠️ "국토교통부_아파트 매매 실거래가 상세 자료" 신청 (반드시 '상세' 버전!)

검색 결과에 비슷한 3개가 나오지만, 우리 앱은 "상세 자료" 버전 전용입니다.
구버전이나 분양권전매 버전을 잘못 신청하면 앱에서 데이터를 못 가져옵니다.

→ ② 번 항목 우측 활용신청 클릭 → 목적: 개인 학습 등 자유 작성 → 신청

3 승인 후 인증키(Service Key) 발급

마이페이지 → 오픈 API → 인증키 발급현황에서 확인 가능

4 아래에 API 키를 입력하여 연결 확인
💡 지역과 날짜는 입력하지 않아도 됩니다. "API 연결 확인" 버튼을 누르면 샘플 지역(강남구)으로 자동 시험합니다. 실제 분석은 "사이클 진단" 탭에서 진행하세요.
📱 휴대폰 카메라로 아래 QR을 스캔하세요
① 휴대폰의 기본 카메라 앱 실행
② QR 코드를 비추면 화면 위에 링크가 뜸
③ 링크 탭 → 휴대폰 브라우저에 자동으로 키가 입력됩니다
⚠️ 이 QR에는 본인 API 키가 들어있습니다. 다른 사람이 보는 화면에서 띄우지 마세요.
⚠️ 주의: CORS 정책으로 인해 브라우저에서 직접 호출이 제한될 수 있습니다. 실제 사용 시에는 백엔드 프록시 서버 또는 CORS 우회 도구가 필요할 수 있어요. 이 앱에서는 API 키 발급 안내 및 연동 준비를 위한 가이드로 활용해주세요.
📍 지역 추가 방법:
지역 추가는 "📈 사이클 진단" 탭의 "관심 지역 관리" 영역에서 합니다.
검색창에 지역명을 입력하거나, 시·도·시·군·구 드롭다운으로 선택하면 전국 250개 시·군·구 코드를 자동으로 불러올 수 있습니다.

❓ 사용 안내 — 처음 사용하시는 분께

이 앱은 국토교통부 실거래가 데이터를 직접 가져옵니다. 무료지만 본인 API 키 발급이 필요해요.

📌 전체 흐름: ① 공공데이터포털 키 발급 (1~2시간) → ② "API 설정" 탭에 키 입력 → ③ 관심 지역 추가 → ④ 진단 실행

1️⃣ 공공데이터 API 키 발급받기 (한 번만)

1 공공데이터포털 접속 및 가입

🌐 www.data.go.kr 접속 → 회원가입 (네이버/카카오 간편로그인 가능)

2 API 검색 — ⚠️ 정확히 "상세 자료" 버전을 골라야 함

상단 검색창에 "아파트 매매 실거래가" 입력 → 비슷한 결과 3개가 보입니다:

① 국토교통부_아파트 매매 실거래가 자료 (구버전 — ❌)
② 국토교통부_아파트 매매 실거래가 상세 자료 (✅ 이걸 선택!)
③ 국토교통부_아파트 분양권전매 실거래가 자료 (다른 용도 — ❌)

※ 우리 앱은 "상세 자료"에만 들어있는 데이터(층/전용면적 등)를 사용합니다.
다른 버전 키를 발급받으면 앱에서 "데이터 없음" 또는 "인증키 오류"가 발생합니다.

② 클릭 → 우측 활용신청 버튼

3 활용 신청

우측 활용신청 버튼 클릭
→ 활용 목적: 개인 학습 등 자유 작성
→ 라이선스 동의 체크 → 신청 완료

4 인증키 확인 (1~2시간 후 자동 승인)

마이페이지 → 오픈 API → 인증키 발급현황
→ 일반 인증키(Encoding) 옆 복사 버튼 클릭

💡 Encoding 키와 Decoding 키 두 개가 보입니다. 반드시 Encoding 키를 복사하세요.

2️⃣ 앱에 API 키 등록

● "🔗 API 설정" 탭으로 이동

상단 탭 메뉴에서 🔗 API 설정 클릭
→ 입력창에 복사한 Encoding 키 붙여넣기
→ 🔗 API 연결 확인 클릭하여 "✅ API 연결 성공" 확인
→ 키는 자동 저장되어 다음에 열 때 다시 입력할 필요 없음

💡 PC ↔ 휴대폰 키 동기화 (QR 이전 기능):
API 키는 기기마다 따로 저장되어, 보통은 PC와 휴대폰에 각각 입력해야 합니다.
이 앱은 QR로 키를 옮기는 기능을 제공합니다:
① PC에서 키 입력 후 "🔗 API 설정" 탭의 "📱 휴대폰으로 가져가기 (QR)" 버튼 클릭
② 휴대폰 기본 카메라 앱으로 화면의 QR 스캔
③ 화면에 뜨는 링크 탭 → 휴대폰에도 자동 입력 완료
⚠️ QR에 키가 들어있으니 본인 또는 가족만 보는 화면에서만 띄우세요.

❓ 자주 묻는 질문

Q. 내 API 키는 안전한가요?

키는 본인의 브라우저(휴대폰/PC)에만 저장됩니다. 외부 서버로 전송되지 않아요.

Q. 일일 호출 한도가 있나요?

공공데이터포털 기본 한도는 하루 10,000건입니다. 일반 사용엔 충분합니다.

Q. 인터넷이 안 될 때도 쓸 수 있나요?

한 번 열어본 적이 있다면 오프라인에서도 앱 화면은 열립니다. 다만 새 실거래 데이터를 받아오려면 인터넷이 필요합니다.

Q. 데이터가 안 보여요

① API 키가 정확히 입력됐는지 (앞뒤 공백 없이)
② 발급 후 1~2시간 지났는지 (즉시 활성화 안 될 수 있음)
③ "🔗 API 설정" 탭의 "API 연결 확인" 버튼으로 먼저 확인

⚠️ 면책: 이 앱은 학습 목적의 참고 도구입니다. 투자 결과의 책임은 본인에게 있습니다.
